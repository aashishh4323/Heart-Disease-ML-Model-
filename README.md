# Heart Disease Prediction — ML Internship Project

**PlutoAcademy AI & ML Internship Program**

## Dataset
UCI Heart Disease Dataset — 1,025 patient records, 14 features  
Source: [Kaggle](https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset)

## Task
Binary Classification: predict presence of heart disease (target = 0 or 1)

## Models Trained
| Model | Accuracy | F1-Score |
|---|---|---|
| Logistic Regression | 80.49% | 44.44% |
| Random Forest | 76.59% | 51.02% |
| KNN (k=13) | 80.49% | 33.33% |

**Best Model: Random Forest** (highest F1-Score — most important for medical data)

## Steps Covered
1. Load, Explore & Preprocess
2. Feature Engineering & Importance
3. Train 3 Models
4. Evaluate & Compare (Accuracy, Precision, Recall, F1)
5. Best Model Analysis, Confusion Matrix & Conclusion
